## Description

Interview project for Binex created by [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install
```

## Copy `env` file

```bash
$ cp .env.example .env
```

## Run application

```bash
# development
$ npm run start:dev

# production
$ npm run build & npm run start:prod
```

## Pre-requirements

MongoDB installation on host machine
